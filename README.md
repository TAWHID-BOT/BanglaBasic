# BanglaBasic
A structured Bangla educational database maintained by **TAWHID ISLAM SIAM**.

## Categories
- পারিভাষিক শব্দ (A-Z)
- বাগধারা (অ-হ)
- এক কথায় প্রকাশ (অ-হ)
- অনুবাদ (অ-হ)
- শব্দার্থ (অ-হ)
- সমার্থক শব্দ (অ-হ)
- বিপরীত শব্দ (অ-হ)
- কবি পরিচিতি
